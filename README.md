CRM project
"CRUD Operations: Manage customer records with GET, POST, PUT/PATCH, and DELETE routes. 
Allow filtering customers by attributes like name, email, or location. 
Implement pagination using query parameters (e.g., page and limit). 
Middlewares: Used for parsing JSON requests and logging HTTP requests. 
Status Codes: Appropriate HTTP status codes for different responses (200, 201, 404, etc.). 
Postman Testing: API is tested using Postman to ensure correct functionality of each route."
